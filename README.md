# 🏎️ Pitstop Planner

Pitstop Planner is a fast, interactive React.js task management application built to demonstrate core state management principles. Themed around the precision and speed of Formula 1 pitstops, this app helps users efficiently track, complete, and clear their daily objectives.

## ✨ Features

* **Add Tasks:** Quickly insert new objectives into your daily lineup.
* **Mark as Completed:** Toggle tasks as done to keep track of your progress.
* **Delete Tasks:** Remove objectives from the grid once they are no longer needed.
* **Responsive State:** Instantaneous UI updates leveraging React's virtual DOM.

## 🛠️ Tech Stack

* **Frontend:** React.js (v18)
* **Build Tool:** Vite
* **Styling:** External CSS (`index.css`)
* **Package Manager:** npm

🚀 Getting Started
Follow these instructions to get a local copy up and running on your machine.

Prerequisites
Node.js is installed on your machine.

Installation
Clone the repository:

Bash: 
    git clone [https://github.com/yourusername/pitstop-planner.git](https://github.com/yourusername/pitstop-planner.git)
    Navigate to the project directory:

Bash
    cd pitstop-planner
    Install dependencies:

    Bash
    npm install
Start the development server:

Bash
npm run dev
Open your browser and visit the local host link provided in your terminal (usually http://localhost:5173/).
